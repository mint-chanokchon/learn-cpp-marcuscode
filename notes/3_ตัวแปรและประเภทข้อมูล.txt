
    * ใช้เพื่อเก็บข้อมูลในหน่วยความจำและช่วยให้เราสามารถจัดการกับข้อมูลได้อย่างง่าย

    * มีประเภทข้อมูลอยู่ 4 ประเภท คือ
        > Character types 
            - เก็บข้อมูลตัวอักษรในรูปแบบ ASCII โดยในภาษา c++ จะมีทั้งหมด 256 ตัวอักษร
            - ค่าของ char ใช้ได้ตั้งแต่ 0 - 255
            > ประเภทข้อมูลของ char
                - char          > 1 byte
                - char16_t      > 2 bytes
                - char32_t      > 4 bytes
                
        > Numerical integer types
            - เป็นค่าที่เก็บข้อมูลจำนวนเต็ม
            - ไม่มีทศนิยม
            > ประเภทข้อมูลของ integer
                - short int     > 2 bytes
                - int           > 2 bytes
                - long int      > 4 bytes
                - long long int > 8 bytes
                - อันที่เป็น unsigned คือไม่สามารถติดลบได้

        > Floating-point types
            > ใช้สำหรับเก็บเลขจำนวนจริงหรือตัวเลขทศนิยม
            > มีอยู่ 2 ประเภท คือ 
                - float         > 4 bytes
                - double        > 8 bytes
                                    
        > Boolean type
            - เก็บค่า จริง เท็จ

        > Strings types
            - เป็นตัวอักษรที่เรียงต่อกันที่แสดงในรูปแบบของ อาเรย์
            - ใน c++ เป็นออปเจ็คของคลาสที่มากับ c++ ไลบรารี่ โดยเราจำเป็นต้องนำเข้าไลบรารี่ของ String
            ด้วยคำสั่ง #include <string>
